# Angular打包流程

  • 新建项目 ng new 项目名
  • cd 项目的根目录下
  • ng generate library 包名称 --prefix=lib
  • 在项目的根目录下找到 angular.json 文件
  • 在 angular.json 文件中 找到 projects 节点 然后再 projects 节点中 找到你创建的 包 然后在包的 architect/build 添加如下配置信息
{
  
  "configurations": {
    "production": {
       "project": "projects/包名称/ng-package.json"
     }
  }
}
  • 在项目的根目录下面 找到 package.json 文件
  • 在 package.json文件中的 Scripts 节点下面 配置如下信息
{
     "build_lib": "ng build --prod 包名称",
     "npm_pack": "cd dist/包名称 && npm pack",
     "package": "npm run build_lib && npm run npm_pack"
}
  • 在dos命令中输入 npm run package 命令执行打包操作
  • 新建组件 ng generate component components/archives --project=包名称(Core)
  • 新建服务 ng generate service services/archives --project=包名称(Core)
  • 打包成tgz包 在别的项目中引用 npm install ./library/包名称(Core)-0.0.1.tgz
